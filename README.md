# CuraHealthAutomation1

Project Overview

Automated end-to-end testing of the CURA Healthcare demo site (https://katalon-demo-cura.herokuapp.com/) using Selenium WebDriver, TestNG, and Java. The automation covers login validation, appointment booking, and error message verification to ensure web application functionality and reliability.

Features Automated

Login Tests

Valid login with correct credentials

Invalid login validation with error message capture

Appointment Booking

Selecting facility from dropdown

Handling checkboxes and radio buttons

Entering visit date and comments

Booking appointment and validating confirmation messages

Browser Management

Launching Chrome browser using WebDriverManager

Independent test execution with @BeforeMethod and @AfterMethod

Parallel test execution capability with TestNG

Technologies & Tools

Java – Programming language

Selenium WebDriver – Web automation framework

TestNG – Test execution and reporting

WebDriverManager – Automatic driver management

Maven – Project build and dependency management

GitHub – Version control and repository hosting
